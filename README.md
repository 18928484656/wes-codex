# LEDUSS Next Website

Factory-style B2B homepage for LEDUSS, built with Next.js.

## Run Locally

If Node.js and npm are installed:

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Current Preview

The development server is running at:

```text
http://127.0.0.1:3000
```

## Main Files

- `app/page.jsx` - homepage content and sections
- `app/layout.jsx` - SEO metadata and page shell
- `app/globals.css` - global styling and responsive layout
- `public/assets/` - website images
